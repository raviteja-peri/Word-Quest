software licensing rules apply abstract this document provides techniques for satisfying the checkpoints defined in user agent accessibility guidelines  uaag these techniques address key aspects of the accessibility of user interfaces content rendering application programming interfaces apis and languages such as the hypertext markup language html cascading style sheets css and the synchronized multimedia integration language smil the techniques listed in this document are not required for conformance to the guidelines these techniques are not necessarily the only way of satisfying the checkpoint nor are they a definitive set of requirements for satisfying a checkpoint status of this document this section describes the status of this document at the time of its publication other
